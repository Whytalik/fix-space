# Специфікація API Contract (FIX Space)

Цей документ визначає повну структуру ендпоінтів REST API для платформи **FIX Space**.

> [!NOTE]
> Детальна інтерактивна документація з точними схемами запитів/відповідей генерується автоматично через OpenAPI (Swagger) і доступна в локальному середовищі за адресою:
> **`http://localhost:3000/api/docs`**

---

## 1. Загальні конвенції

| Параметр           | Опис                                                                                      |
| :----------------- | :---------------------------------------------------------------------------------------- |
| **Base URL**       | `http://localhost:3000` (для локальної розробки)                                          |
| **Формат даних**   | JSON (`Content-Type: application/json`)                                                   |
| **Автентифікація** | JWT (через HTTP-only cookie `access_token` або заголовок `Authorization: Bearer <token>`) |
| **Ідентифікатори** | Усі ID генеруються як UUID v4                                                             |
| **Формат часу**    | ISO 8601 UTC (`YYYY-MM-DDTHH:mm:ss.sssZ`)                                                 |

### Стандартна структура помилки (Error Response)

```json
{
  "message": "Опис помилки або масив помилок валідації",
  "statusCode": 400,
  "timestamp": "2026-06-06T12:00:00.000Z"
}
```

| `POST` | `/auth/logout`              | Вихід із системи та видалення сесії                   | Так  | — (видаляє `refresh_token` з cookies)         |
