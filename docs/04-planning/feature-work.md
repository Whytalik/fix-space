# Feature Work — v2

Фічі, що не увійшли в v1, але заплановані на майбутнє.

---

## 1. Скидання демо-даних

- Скидання демо-workspace до початкового стану (відновлення seed-даних), щоб почати роботу з чистого аркуша.

## 2. Різні види подання записів

Крім табличного, додати альтернативні відображення:

- **Board / Kanban** — картки, згруповані по колонках статусу
- **Gallery** — сітка карток з прев'ю
- **Calendar** — записи на календарній сітці за датами

## 3. Button (кнопка як тип властивості)

Новий тип властивості, що дозволяє виконувати дії прямо з таблиці чи запису:

- Кнопка-колонка в таблиці, швидка дія одним кліком
- Дія: встановити значення поля, створити запис
- Кнопка як блок у вмісті запису
- Діалог підтвердження дії

## 4. Кошик

М'яке видалення записів з можливістю відновлення протягом 30 днів. Автоматичне очищення після завершення терміну зберігання.

## 5. Керування Google-акаунтом

- Встановлення пароля в налаштуваннях для користувача, зареєстрованого через Google (після цього доступний вхід за email/password)
- Відв'язка Google-акаунту в налаштуваннях (доступно лише якщо встановлено пароль — не можна залишити акаунт без жодного методу входу)
