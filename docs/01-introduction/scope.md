# Scope Document — FIX Space v1.0

Визначає що входить і що явно не входить до першої публічної версії платформи. Без цього документа обсяг роботи розростається непомітно.

---

## Контекст

FIX Space — веб-платформа для організації діяльності CFD-трейдера. Система не виконує торгових операцій і не є торговим терміналом. Її задача — централізоване зберігання, структурування та аналіз торгових даних у єдиному середовищі.

Цільовий користувач v1.0: приватний CFD/forex/крипто-трейдер, що торгує самостійно і хоче вести торговий журнал, фіксувати помилки, аналізувати статистику та систематизувати свій досвід.

---

## Що входить у v1.0

### Автентифікація та акаунт

- Реєстрація, підтвердження email, вхід, вихід
- Скидання та зміна пароля
- Управління профілем (нікнейм, аватар, email)
- Перегляд активних сесій та їх завершення
- Видалення акаунту

### Робочий простір

- Автоматичне створення простору зі стартовим пакетом баз даних при реєстрації
- Домашня сторінка з індикатором торгових сесій, щоденним воркфлоу, сьогоднішніми записами та міні-графіками статистики
- Створення, редагування, дублювання, видалення просторів
- Перемикання між просторами (до 5 на акаунт)

### Секції та навігація

- Групування баз даних у іменовані секції
- Зміна порядку секцій та баз даних
- Згортання/розгортання секцій у сайдбарі

### Бази даних

- Стартовий набір пресетних баз: Trading Journal, Daily Routine, Notes, Mistakes, Accounts, Operations, Trading System
- Створення власних баз даних
- Дублювання, блокування структури, видалення (пресетні не можна видалити)
- Підсумкові показники по стовпцях (сума, середнє, медіана тощо)

### Властивості (типи даних)

- TEXT, NUMBER, DATE, CHECKBOX, DURATION
- SELECT (одиночний і множинний вибір), STATUS
- RELATION (зв'язки між базами)
- FORMULA (обчислювані поля через візуальний конструктор)
- RATING, PROGRESS
- BUTTON (дія без збереження значення)

### Записи

- Створення, перегляд, редагування, видалення записів
- Дублювання записів
- Масове редагування та видалення
- Кошик з утриманням 30 днів та можливістю відновлення

### Контентна область запису

- Вільна зона для довільного контенту: текст, зображення, таблиці, чеклісти, відео та 23 типи компонентів
- Ієрархія: рядки → колонки → блоки → компоненти
- Бібліотека готових блоків (системні пресети + власні збережені)
- Undo/redo, автоматичне збереження, знімки (history) з відновленням

### Шаблони

- Системні пресети для кожної бази + власні шаблони
- Name Pattern (автогенерація назви запису за токенами: дата, місяць, лічильник тощо)
- Встановлення шаблону за замовчуванням для бази

### Подання (Views)

- Табличний тип відображення
- Фільтрація з логікою AND/OR, іменовані фільтр-пресети
- Сортування за кількома критеріями одночасно
- Групування за властивістю або метаполем
- Показ/приховування стовпців, регулювання ширини
- Пагінація (10 / 25 / 50 / 75 / 100 записів)
- До 10 збережених подань на базу

### Пошук

- Глобальний пошук по всьому простору (Ctrl+K)
- Пошук в межах бази за назвою та значеннями властивостей

### Імпорт та Експорт

- Імпорт записів із CSV з маппінгом полів та попереднім переглядом
- Експорт записів у CSV (з урахуванням активних фільтрів)

### Інтеграції із зовнішніми сервісами

- Binance, Bybit, OKX (через read-only API-ключі; авто-синк кожні 5 хвилин)
- MetaTrader 5 (через Investor Password; read-only)
- cTrader (через OAuth 2.0)
- Ручний запуск синхронізації та вибір діапазону для початкового завантаження

### Формули

- Галерея пресетних обчислень (R-Multiple, Запланований RR, Ризик % від балансу, Дотримання правил тощо)
- Покроковий конструктор власних розрахунків через дропдауни (без ручного введення синтаксису)

### Автоматизації

- Конструктор правил: тригер → опціональна умова → дія
- Виконання у фоновому режимі
- До 10 правил на базу даних

### Статистика

- Торгова статистика на основі пресетних баз (Win Rate, Profit Factor, RR-крива, drawdown тощо)
- Власні звіти для будь-якої підключеної бази (авто-генерація графіків за схемою)
- Режим порівняння двох часових діапазонів

### Налаштування

- Профіль, безпека, зовнішній вигляд (тема, мова)
- Формат дат/часу, часовий пояс, початок тижня
- Налаштування значень за замовчуванням для секцій, баз даних, записів
- Управління інтеграціями

### Онбординг

- Демо-простір із готовими даними
- Покроковий тур по інтерфейсу
- Чеклист активації
- Контекстні підказки

### Сповіщення

- In-app сповіщення (дзвіночок у сайдбарі)
- Сповіщення про статус автоматизацій та інтеграцій

## Що явно НЕ входить у v1.0

| Функція                                   | Причина виключення                                                                                             |
| ----------------------------------------- | -------------------------------------------------------------------------------------------------------------- |
| **Мобільний застосунок** (iOS / Android)  | Тільки веб-платформа; мобільний адаптивний вигляд — не гарантується                                            |
| **IBKR (Interactive Brokers) інтеграція** | Орієнтований на institutional трейдерів, поза межами цільової аудиторії; архітектурно передбачений на майбутнє |
| **Kanban та Calendar подання**            | Лише табличне подання в v1.0; інші типи View — архітектурно передбачені                                        |
| **Спільні робочі простори**               | Кожен простір ізольований і належить одному користувачу; немає спільного доступу, коментарів чи ролей          |
| **AI-аналіз та рекомендації**             | Жодних ML/AI функцій у v1.0                                                                                    |
| **Двофакторна автентифікація (2FA)**      | Стандартна JWT-автентифікація; 2FA відсутня                                                                    |
| **Публічний API / вебхуки**               | Немає зовнішнього API для сторонніх інтеграцій                                                                 |
| **Офлайн-режим**                          | Потрібне постійне підключення до інтернету                                                                     |
| **Сповіщення поза застосунком**           | Тільки in-app; без push, email або SMS сповіщень про події                                                     |
| **Резервне копіювання до хмари**          | Немає інтеграції з Google Drive, Dropbox тощо                                                                  |
| **Публічні профілі та соціальні функції** | Платформа суто приватна                                                                                        |
