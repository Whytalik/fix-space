### [ ] TC-STAT-U-001: StatisticsService — розрахунок ключових торгових метрик (Win Rate, Profit Factor, Expectancy)

| Поле         | Значення          |
| ------------ | ----------------- |
| **US**       | US-026            |
| **Issue**    | #45               |
| **TS**       | —                 |
| **Метод**    | Unit (Jest)       |
| **Техніка**  | Equivalence Part. |
| **Priority** | P1                |

**Кроки:**

1. Передати в `calculateTradingMetrics` масив угод (7 виграшних із загальним профітом $700, 3 програшні із загальним збитком $300, середня комісія $5 на угоду).
2. Передати масив угод із 0 збитків (для перевірки ділення на 0 при розрахунку Profit Factor).
3. Передати пустий масив угод.

**Очікуваний результат:**

- Крок 1: Win Rate = 70%, Profit Factor = 700 / 300 = 2.33, Expectancy = (700 - 300 - 50) / 10 = $35.
- Крок 2: Profit Factor = Infinity (або коректно оброблене значення, наприклад, рівне сумарному прибутку або null).
- Крок 3: всі метрики дорівнюють 0, система не падає з помилкою.
