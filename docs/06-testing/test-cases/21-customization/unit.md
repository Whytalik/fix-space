| ------------ | ---------------- |

| **Техніка**  | State Transition |
| **Priority** | P2               |

**Кроки:**

1. Записати початковий стан контенту `State0`.
2. Внести зміни (додати компонент), записати `State1`.
3. Викликати `undo(recordId)` та перевірити поточний стан.
