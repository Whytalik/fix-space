## 21. Customization (TC-CUST-001 — TC-CUST-004)

---

### TC-CUST-001: Збереження блоку до бібліотеки

| Поле         | Значення            |
| ------------ | ------------------- |
| **US**       | US-136              |
| **Issue**    | #116                |
| **TS**       | —                   |
| **Метод**    | Manual (Browser)    |
| **Техніка**  | Exploratory Testing |
| **Priority** | P3                  |

**Кроки:**

1. Відкрити контекстне меню блоку
2. Обрати "Save to Library"
3. Перевірити що блок з'явився в бібліотеці

**Очікуваний результат:**

- Блок збережено в особистій бібліотеці

### TC-CUST-003: Діалог підтвердження видалення розділу — перемикач

| Поле         | Значення            |
| ------------ | ------------------- |
| **US**       | US-137              |
| **Issue**    | #122                |
| **TS**       | —                   |
| **Метод**    | Manual (Browser)    |
| **Техніка**  | Exploratory Testing |
| **Priority** | P3                  |

**Кроки:**

1. Settings → Sections → "Show delete confirmation"
2. Вимкнути перемикач
3. Видалити розділ — перевірити що підтвердження не з'являється

**Очікуваний результат:**

- Розділ видалено без діалогу підтвердження
