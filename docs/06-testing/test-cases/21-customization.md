### [ ] TC-CUST-U-003: LibraryBlockService — оновлення версії блоку бібліотеки та оновлення копій

| Поле         | Значення        |
| ------------ | --------------- |
| **US**       | US-136          |
| **Issue**    | #116            |
| **TS**       | —               |
| **Метод**    | Unit (Jest)     |
| **Техніка**  | Branch Coverage |
| **Priority** | P3              |

**Кроки:**

1. Оновити бібліотечний блок з опцією `Update everywhere` та перевірити копії у записах.
2. Оновити блок з опцією `Update template` та перевірити копії.

- Крок 3: зв'язок запису із бібліотечним блоком видалено (`libraryBlockId = null`), контент збережено як локальну копію запису.
