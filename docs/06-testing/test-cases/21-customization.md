### TC-CUST-005: Зміна візуальної кастомізації (обкладинка, іконки) бази даних (Playwright UI E2E)

| Поле         | Значення            |
| ------------ | ------------------- |
| **US**       | US-136, US-137      |
| **Issue**    | #116, #122          |
| **TS**       | —                   |
| **Метод**    | E2E UI (Playwright) |
| **Техніка**  | Exploratory Testing |
| **Priority** | P3                  |

**Кроки:**

1. Увійти в систему та відкрити базу даних Trading Journal.
2. Натиснути на поточну іконку бази даних у заголовку сторінки.
3. У спливаючому вікні вибору емодзі змінити іконку на `"💼"` (портфель).
4. Натиснути кнопку "Додати обкладинку" (Add Cover) над заголовком бази даних.
5. У галереї обкладинок вибрати градієнтний пресет або колір та зберегти.
6. Перезавантажити сторінку браузера.
7. Перевірити, що обрана іконка `"💼"` та обрана обкладинка відображаються коректно у заголовку та боковому меню.

**Очікуваний результат:**

- Візуальні елементи кастомізації бази даних успішно оновлюються в UI та зберігаються після оновлення сторінки.

### TC-CUST-006: Зміна акцентного кольору інтерфейсу через налаштування (Playwright)

| Поле         | Значення                              |
| ------------ | ------------------------------------- |
| **US**       | US-136                                |
| **Issue**    | —                                     |
| **TS**       | —                                     |
| **Метод**    | E2E UI (Playwright)                   |
| **Техніка**  | Equivalence Partitioning (valid path) |
| **Priority** | P3                                    |

**Кроки:**

1. Увійти в систему та перейти до розділу налаштувань Customization
2. Знайти палітру акцентних кольорів та вибрати колір відмінний від поточного
3. Підтвердити вибір або зберегти (залежно від реалізації)
4. Перевірити що кнопки та інтерактивні елементи відображають новий акцентний колір
5. Оновити сторінку та перевірити збереження кольору

**Очікуваний результат:**

- Акцентний колір застосовується до UI-елементів миттєво
- Вибраний колір зберігається після оновлення сторінки
