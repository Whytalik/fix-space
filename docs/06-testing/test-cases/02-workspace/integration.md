### [x] TC-WS-001: Автоматичне створення workspace при реєстрації

| Поле         | Значення                 |
| ------------ | ------------------------ |
| **US**       | US-004                   |
| **Issue**    | #6                       |
| **TS**       | TS-AUTH-01               |
| **Метод**    | Integration (Supertest)  |
| **Техніка**  | Equivalence Partitioning |
| **Priority** | P1                       |

**Кроки:**

1. Зареєструвати нового користувача
2. Верифікувати email
3. Увійти в систему
4. `GET /spaces` з Authorization header
5. Перевірити кількість та наявність пресетних баз

**Очікуваний результат:**

- Статус: `200 OK`
- Один workspace створено автоматично
