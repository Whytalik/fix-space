### [ ] TC-WS-022: Створення та перемикання Workspace (Playwright UI E2E)

| Поле         | Значення                              |
| ------------ | ------------------------------------- |
| **US**       | US-011, US-012                        |
| **Issue**    | #35, #36                              |
| **TS**       | TS-SPACE-01                           |
| **Метод**    | E2E UI (Playwright)                   |
| **Техніка**  | Equivalence Partitioning (valid path) |
| **Priority** | P1                                    |

**Кроки:**

1. Увійти в систему
2. Клацнути на селектор просторів (Workspace Selector) у лівому меню
3. Натиснути кнопку "+ Створити простір"
4. Ввести назву `"Дипломний Простір"` у спливаючому модальному вікні
5. Натиснути "Створити"
6. Перевірити, що назва простору оновилася в лівому меню
7. Клацнути на назву простору і перемкнутися назад на дефолтний простір

**Очікуваний результат:**

- Новий простір успішно створюється в базі даних та відображається в інтерфейсі
- При натисканні назва простору миттєво змінюється в лівому меню без перезавантаження
- Списки пресетних баз даних створюються автоматично та коректно завантажуються під кожен простір

**Кроки:**
