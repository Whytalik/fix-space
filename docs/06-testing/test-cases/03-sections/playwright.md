### [ ] TC-SEC-015: Створення та відображення нової секції в боковому меню (Playwright UI E2E)

| Поле         | Значення                              |
| ------------ | ------------------------------------- |
| **US**       | US-018                                |
| **Issue**    | #38                                   |
| **TS**       | TS-SPACE-01                           |
| **Метод**    | E2E UI (Playwright)                   |
| **Техніка**  | Equivalence Partitioning (valid path) |
| **Priority** | P1                                    |

**Кроки:**

1. Увійти в систему
2. У лівому меню (Sidebar) натиснути на кнопку "+ Додати секцію" (Add Section)
3. У модальному вікні ввести назву секції `"Аналітика та стратегії"`
4. Вибрати іконку `"📈"` та колір `"blue"`
5. Натиснути "Зберегти"
6. Перевірити, що нова секція відображається в боковому меню з правильним текстом та іконкою

**Очікуваний результат:**

- Секція створюється в базі даних та відображається в боковій панелі
- Всі іконки та назва секції рендеряться миттєво без потреби перезавантаження

### [ ] TC-SEC-016: Перейменування секції через UI (Playwright)

| Поле         | Значення                              |
| ------------ | ------------------------------------- |
| **US**       | US-019                                |
| **Issue**    | —                                     |
| **TS**       | —                                     |
| **Метод**    | E2E UI (Playwright)                   |
| **Техніка**  | Equivalence Partitioning (valid path) |
| **Priority** | P2                                    |

**Кроки:**

1. Увійти в систему та відкрити робочий простір зі створеною секцією
2. Двічі клікнути на назву секції в боковому меню або вибрати "Rename" з контекстного меню
3. Очистити поточну назву та ввести нову `"Перейменована секція"`
4. Підтвердити зміну клавішею Enter або кліком поза полем введення
5. Перевірити оновлену назву в боковій панелі

**Очікуваний результат:**

- Назва секції миттєво оновлюється в боковому меню без перезавантаження сторінки
- Нова назва зберігається та відображається коректно після оновлення сторінки
