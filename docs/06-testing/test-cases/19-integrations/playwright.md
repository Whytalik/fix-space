# Playwright UI E2E Tests: Integrations

### [x] TC-INT-015: Перевірка інтерфейсу підключення та конфігурації інтеграцій (Playwright UI E2E)

| Поле         | Значення            |
| ------------ | ------------------- |
| **US**       | US-075, US-081      |
| **Issue**    | #105, #109          |
| **TS**       | —                   |
| **Метод**    | E2E UI (Playwright) |
| **Техніка**  | Exploratory Testing |
| **Priority** | P3                  |

**Кроки:**

1. Увійти в систему та перейти на сторінку "Налаштування" (Settings) -> "Інтеграції" (Integrations).
2. Перевірити наявність списку доступних платформ (Binance, MT5).
3. Натиснути кнопку "Підключити" (Connect) біля провайдера Binance.
4. У діалоговому вікні ввести тестові API Key та API Secret.
5. Натиснути кнопку "Зберегти" (Save).
6. Спостерігати за оновленням статусу підключення.
7. Переконатися, що підключений акаунт з'явився у списку інтеграцій зі статусом `"Connected"` та часом останньої синхронізації.
8. Натиснути кнопку "Відключити" (Disconnect) біля Binance та підтвердити дію у поп-апі.
9. Переконатися, що інтеграцію успішно видалено зі списку.

**Очікуваний результат:**

- Інтерфейс інтеграцій дозволяє підключати, перевіряти статус та коректно відключати акаунти зовнішніх бірж без помилок в UI.
