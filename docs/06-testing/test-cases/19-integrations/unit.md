| ------------ | ------------------ |
| **US**       | US-075             |
| **Issue**    | —                  |
| **TS**       | —                  |
| **Метод**    | Unit (Jest)        |
| **Техніка**  | Statement Coverage |
| **Priority** | P1                 |

**Кроки:**

1. Налаштувати мок репозиторію або Prisma клієнта на успішне виконання першої частини транзакції (наприклад, створення запису).
