### [ ] TC-AUTO-014: Створення та спрацьовування автоматизації в реальному часі (Playwright UI E2E)

| Поле         | Значення            |
| ------------ | ------------------- |
| **US**       | US-036, US-036      |
| **Issue**    | #59                 |
| **TS**       | —                   |
| **Метод**    | E2E UI (Playwright) |
| **Техніка**  | Exploratory Testing |
| **Priority** | P2                  |

**Кроки:**

1. Увійти в систему та відкрити розділ "Автоматизація" (Automations) в налаштуваннях бази даних Trading Journal.
2. Натиснути кнопку "+ Створити автоматизацію" та обрати ручний режим налаштування.
3. Вказати тригер: Зміна значення властивості `Result` (Результат) на `"Loss"` (Збиток).
4. Вказати дію: Автоматично встановити для властивості `Mistakes` (Помилки) значення `"FOMO"`.
5. Зберегти автоматизацію та переконатися, що вона з'явилася в списку та є активною.
6. Перейти до таблиці Trading Journal та знайти угоду з порожніми полями `Result` та `Mistakes`.
7. Змінити значення `Result` на `"Loss"` і зберегти.
8. Переконатися, що значення в полі `Mistakes` автоматично та в реальному часі оновилося на `"FOMO"`.

**Очікуваний результат:**

- Автоматизація успішно створюється в UI.
- Після тригерування умови, дія виконується миттєво та оновлює пов'язане поле у відкритій таблиці без перезавантаження.

### [ ] TC-AUTO-015: Увімкнення та вимкнення автоматизації через UI (Playwright)

| Поле         | Значення            |
| ------------ | ------------------- |
| **US**       | US-037              |
| **Issue**    | —                   |
| **TS**       | —                   |
| **Метод**    | E2E UI (Playwright) |
| **Техніка**  | State Transition    |
| **Priority** | P3                  |

**Кроки:**

1. Відкрити налаштування бази даних → секція Automations
2. Знайти наявну увімкнену автоматизацію
3. Перемкнути тумблер "Active" в стан вимкнено
4. Перевірити що статус автоматизації змінився на "Inactive"
5. Повторно увімкнути та перевірити зворотній стан

**Очікуваний результат:**

- Статус автоматизації перемикається між Active/Inactive без перезавантаження сторінки
- Вимкнена автоматизація не виконує дії при спрацьовуванні умови
