### [ ] TC-IMP-U-001: CsvParser — розбір CSV-файлу, детекція кодування та валідація розміру

| Поле         | Значення       |
| ------------ | -------------- |
| **US**       | US-087         |
| **Issue**    | #49            |
| **TS**       | —              |
| **Метод**    | Unit (Jest)    |
| **Техніка**  | Boundary Value |
| **Priority** | P2             |

**Кроки:**

1. Передати в метод парсингу файл розміром 26 МБ.
2. Передати файл з різними роздільниками (кома, крапка з комою, табуляція).
3. Передати файл у кодуванні UTF-8 із кириличними символами.

**Очікуваний результат:**

- Крок 2: парсер автоматично розпізнає роздільник та розділяє рядки на масив колонок.
- Крок 3: парсер успішно декодує кириличний текст без спотворення символів.
