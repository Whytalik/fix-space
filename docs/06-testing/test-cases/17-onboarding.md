### TC-ONB-012: Проходження вітального туру нового користувача при першому вході (Playwright UI E2E)

| Поле         | Значення            |
| ------------ | ------------------- |
| **US**       | US-126              |
| **Issue**    | #100                |
| **TS**       | —                   |
| **Метод**    | E2E UI (Playwright) |
| **Техніка**  | Exploratory Testing |
| **Priority** | P2                  |

**Кроки:**

1. Увійти вперше під тестовим обліковим записом нового користувача.
2. Переконатися, що після першого завантаження сторінки автоматично з'являється перший поп-ап (крок туру) з описом інтерфейсу та кнопкою "Далі".
3. Натиснути кнопку "Далі" для переходу на наступні кроки.
4. Пройти 3 кроки туру, спостерігаючи, як підсвічуються відповідні зони інтерфейсу (Sidebar, кнопки додавання записів, простір таблиці).
5. На 4-му кроці натиснути "Пропустити тур".
6. Переконатися, що модальні підказки зникли з екрану та не з'являються повторно після оновлення сторінки.

**Очікуваний результат:**

- Інтерактивний вітальний тур коректно відображає підказки відповідно до елементів UI.
- Після закриття чи пропуску туру його стан зберігається, і інтерфейс більше не блокує користувача.

### TC-ONB-013: Повний флоу онбордингу нового користувача (Playwright)

| Поле         | Значення            |
| ------------ | ------------------- |
| **US**       | US-125              |
| **Issue**    | —                   |
| **TS**       | —                   |
| **Метод**    | E2E UI (Playwright) |
| **Техніка**  | State Transition    |
| **Priority** | P2                  |

**Кроки:**

1. Зареєструвати нового користувача через API та верифікувати email
2. Увійти в систему — перевірити відображення онбординг-кроків або вітального туру
3. Послідовно виконати всі онбординг-кроки або натиснути "Skip / Пропустити"
4. Перевірити що після завершення відображається основний дашборд
5. Оновити сторінку та переконатися що онбординг не показується повторно

**Очікуваний результат:**

- Онбординг завершується без помилок та переходить до дашборду
- При повторному вході онбординг не відображається (стан збережено)
