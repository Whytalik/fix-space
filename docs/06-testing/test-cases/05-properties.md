### [ ] TC-PROP-001: Створення властивості — TEXT

| Поле         | Значення                         |
| ------------ | -------------------------------- |
| **US**       | US-032                           |
| **Issue**    | #9                               |
| **TS**       | TS-PROP-01                       |
| **Метод**    | Black Box (Postman)              |
| **Техніка**  | Equivalence Partitioning (valid) |
| **Priority** | P1                               |

**Кроки:**

1. Увійти в систему, отримати databaseId
2. `POST /properties` з тілом:
   ```json
   {
     "databaseId": "<db-id>",
     "name": "Notes",
     "type": "TEXT",
     "config": { "isRichText": true }
   }
   ```
3. Перевірити статус-код

**Очікуваний результат:**

- Статус: `201 Created`
- Властивість створена з типом TEXT

3. Клікнути на іконку видалення або вибрати "Delete" з меню властивості
4. Підтвердити видалення у діалоговому вікні
5. Перевірити що колонка зникла з таблиці

**Очікуваний результат:**

- Властивість видаляється зі списку та зникає як колонка у Table View
- Після оновлення сторінки колонка відсутня та дані в БД видалені
