| ------------ | ------------------------------------------- |

**Кроки:**

### [ ] TC-AUTH-028: Успішне завантаження сторінки входу (Playwright UI E2E)

| ------------ | ------------------------------------- |
| **US**       | US-002                                |
| **Issue**    | #3                                    |
| **TS**       | TS-AUTH-03                            |
| **Метод**    | E2E UI (Playwright)                   |
| **Техніка**  | Equivalence Partitioning (valid path) |
| **Priority** | P1                                    |

**Кроки:**

1. Відкрити браузер та перейти на сторінку `/login`
2. Очікувати редірект на локалізований шлях (наприклад, `/en/login` або `/uk/login`)

- Головні елементи інтерфейсу форми входу видимі та доступні

### [ ] TC-AUTH-033: Повний E2E флоу реєстрації та входу (Playwright)

| ------------ | ---------------------- |
| **US**       | US-001, US-002         |
| **Issue**    | #1, #3                 |
| **TS**       | TS-AUTH-01, TS-AUTH-03 |
| **Метод**    | E2E UI (Playwright)    |
| **Техніка**  | State Transition       |
| **Priority** | P1                     |

**Кроки:**

1. Перейти на `/en/register`
2. Заповнити поля: email, username, password
3. Натиснути кнопку "Create account"

**Очікуваний результат:**

- Реєстрація завершується успішно: відображається повідомлення про перевірку email
- Після верифікації та входу URL завершується на `/en`
- Дашборд відображається без помилок
