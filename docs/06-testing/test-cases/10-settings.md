### TC-SET-019: Перемикання теми інтерфейсу (Світла/Темна) (Playwright UI E2E)

| Поле         | Значення            |
| ------------ | ------------------- |
| **US**       | US-120              |
| **Issue**    | #93                 |
| **TS**       | —                   |
| **Метод**    | E2E UI (Playwright) |
| **Техніка**  | State Transition    |
| **Priority** | P2                  |

**Кроки:**

1. Увійти в систему та перейти до розділу "Налаштування" (Settings)
2. Обрати підрозділ "Зовнішній вигляд" (Appearance)
3. Натиснути на перемикач теми та вибрати `"Темна" (Dark)`
4. Перевірити, що на тег `<html>` або `<body>` додано CSS-клас `dark` (або відповідний атрибут темної теми)
5. Перемкнути тему на `"Світла" (Light)` та перевірити зникнення класу `dark`

**Очікуваний результат:**

- Тема інтерфейсу перемикається миттєво та без перезавантаження сторінки
- Класи стилів темної теми застосовуються до всього інтерфейсу системи
