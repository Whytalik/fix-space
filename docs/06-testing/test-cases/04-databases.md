### TC-DB-022: Створення кастомної бази даних у робочому просторі (Playwright UI E2E)

| Поле         | Значення                              |
| ------------ | ------------------------------------- |
| **US**       | US-025                                |
| **Issue**    | #32                                   |
| **TS**       | TS-DB-01                              |
| **Метод**    | E2E UI (Playwright)                   |
| **Техніка**  | Equivalence Partitioning (valid path) |
| **Priority** | P1                                    |

**Кроки:**

1. Увійти в систему та відкрити робочий простір
2. У лівому меню (Sidebar) натиснути на кнопку "+" біля назви секції для додавання нової бази даних
3. У модальному вікні ввести назву бази даних `"Мій торговий щоденник"`
4. Натиснути "Створити"
5. Перевірити, що нова база даних з'явилася в боковому меню
6. Перевірити, що при натисканні на неї відкривається порожня таблиця (з єдиною колонкою Name)

**Очікуваний результат:**

- База даних успішно створюється
- Сторінка миттєво перенаправляється на новостворену базу даних
- Заголовок та бокова панель відображають назву `"Мій торговий щоденник"`
