### [x] TC-SET-019: Перемикання теми інтерфейсу (Світла/Темна) (Playwright UI E2E)

| Поле         | Значення            |
| ------------ | ------------------- |
| **US**       | US-065              |
| **Issue**    | #93                 |
| **TS**       | —                   |
| **Метод**    | E2E UI (Playwright) |
| **Техніка**  | State Transition    |
| **Priority** | P2                  |

**Кроки:**

1. Увійти в систему та перейти до розділу "Налаштування" (Settings)
2. Обрати підрозділ "Зовнішній вигляд" (Appearance)
3. Натиснути на перемикач теми та вибрати `"Темна" (Dark)`
4. Перевірити, що на тег `<html>` або `<body>` додано CSS-клас `dark` (або відповідний атрибут темної теми)
5. Перемкнути тему на `"Світла" (Light)` та перевірити зникнення класу `dark`

**Очікуваний результат:**

- Тема інтерфейсу перемикається миттєво та без перезавантаження сторінки
- Класи стилів темної теми застосовуються до всього інтерфейсу системи

### [x] TC-SET-020: Оновлення відображуваного імені профілю через UI (Playwright)

| Поле         | Значення                              |
| ------------ | ------------------------------------- |
| **US**       | US-025                                |
| **Issue**    | —                                     |
| **TS**       | TS-SET-01                             |
| **Метод**    | E2E UI (Playwright)                   |
| **Техніка**  | Equivalence Partitioning (valid path) |
| **Priority** | P1                                    |

**Кроки:**

1. Увійти в систему та перейти на сторінку Profile (`/profile`)
2. Знайти поле відображуваного імені (Display Name / Username)
3. Очистити поточне значення та ввести нове ім'я `"Оновлений Профіль"`
4. Зберегти зміни натисканням кнопки Save
5. Перевірити оновлене ім'я у навігаційній кнопці header

**Очікуваний результат:**

- Нове ім'я відображається у header додатку без перезавантаження сторінки
- Зміни зберігаються після оновлення сторінки
