### [ ] TC-REC-001: Створення запису з шаблону

| Поле         | Значення                         |
| ------------ | -------------------------------- |
| **US**       | US-005                           |
| **Issue**    | #22                              |
| **TS**       | TS-REC-01                        |
| **Метод**    | Integration (Supertest)          |
| **Техніка**  | Equivalence Partitioning (valid) |
| **Priority** | P1                               |

**Кроки:**

1. Отримати databaseId та templateId
2. `POST /records` з тілом:
   ```json
   { "databaseId": "<db-id>", "templateId": "<template-id>" }
   ```
3. Перевірити статус-код
4. Перевірити що запис має значення властивостей з шаблону

**Очікуваний результат:**

- Статус: `201 Created`
- Назва запису згенерована з патерну шаблону
- Властивості мають початкові значення з шаблону
- Контентна область скопійована з шаблону
