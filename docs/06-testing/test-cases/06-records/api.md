### [ ] TC-REC-027: Метаполя — фільтрація за Дата створення

| Поле         | Значення                 |
| ------------ | ------------------------ |
| **US**       | US-046                   |
| **Issue**    | #189                     |
| **TS**       | —                        |
| **Метод**    | Manual (Postman)         |
| **Техніка**  | Equivalence Partitioning |
| **Priority** | P2                       |

**Кроки:**

1. Створити 3 записи в різний час
2. `GET /records?databaseId=<id>&filter[createdAt][gte]=<timestamp>` з конкретною датою
3. Перевірити що повернуто лише записи, створені після вказаного timestamp

**Очікуваний результат:**

- Статус: `200 OK`
- Відповідь містить лише записи з `createdAt >= timestamp`
- Кожен запис має поля `createdAt` та `updatedAt` у відповіді
