### [x] TC-FORM-U-001: FormulaEvaluator — розрахунок пресету "Запланований RR"

| Поле         | Значення       |
| ------------ | -------------- |
| **US**       | US-009         |
| **Issue**    | #19            |
| **TS**       | —              |
| **Метод**    | Unit (Jest)    |
| **Техніка**  | Boundary Value |
| **Priority** | P1             |

**Кроки:**

1. Обчислити RR для Long угоди: Ціна входу = 100, Стоп-лосс = 90, Ціна цілі = 130.
2. Обчислити RR для Short угоди: Ціна входу = 100, Стоп-лосс = 110, Ціна цілі = 70.

**Очікуваний результат:**

- Крок 1: RR = (130 − 100) / (100 − 90) = 3.0 (number).
- Крок 2: RR = (100 − 70) / (110 − 100) = 3.0 (number).
- Крок 3: значення RR від'ємне (помилка), повертається `null` або маркується як помилка конфігурації.
