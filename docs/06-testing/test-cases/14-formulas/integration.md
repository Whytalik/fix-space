### [ ] TC-FORM-001: FORMULA — автоматичний розрахунок Net PnL

| Поле         | Значення                 |
| ------------ | ------------------------ |
| **US**       | US-009                   |
| **Issue**    | #19                      |
| **TS**       | —                        |
| **Метод**    | Integration (Supertest)  |
| **Техніка**  | Equivalence Partitioning |
| **Priority** | P1                       |

**Кроки:**

1. Створити запис в Trading Journal з PnL=100, Commissions=5, Fees=2, Swap=1
2. Перевірити що Net PnL розраховано

**Очікуваний результат:**

- Net PnL = 100 - 5 - 2 - 1 = 92
