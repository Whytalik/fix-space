### TC-TMPL-016: Створення та застосування кастомного шаблону запису (Playwright UI E2E)

| Поле         | Значення            |
| ------------ | ------------------- |
| **US**       | US-068              |
| **Issue**    | #87                 |
| **TS**       | TS-TMPL-01          |
| **Метод**    | E2E UI (Playwright) |
| **Техніка**  | Exploratory Testing |
| **Priority** | P2                  |

**Кроки:**

1. Увійти в систему та відкрити існуючу базу даних.
2. Відкрити будь-який запис, наповнити його вмістом (наприклад, додати текстовий блок) та змінити декілька властивостей.
3. У меню опцій запису натиснути "Створити шаблон з цього запису".
4. У діалоговому вікні ввести назву нового шаблону `"Мій кастомний шаблон"` та зберегти.
5. Створити новий порожній запис у цій же базі даних.
6. У списку доступних шаблонів обрати `"Мій кастомний шаблон"`.
7. Перевірити, що всі заповнені властивості та вміст автоматично скопіювалися у новий запис.

**Очікуваний результат:**

- Новий кастомний шаблон успішно створюється та з'являється у списку шаблонів.
- При його застосуванні властивості та контент нового запису заповнюються автоматично згідно з шаблоном.
