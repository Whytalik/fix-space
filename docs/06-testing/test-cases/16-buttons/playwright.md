### [ ] TC-BTN-009: Клік по кнопці швидкої дії та спрацьовування тригера (Playwright UI E2E)

| Поле         | Значення            |
| ------------ | ------------------- |
| **US**       | US-108, US-111      |
| **Issue**    | #61, #118           |
| **TS**       | —                   |
| **Метод**    | E2E UI (Playwright) |
| **Техніка**  | Exploratory Testing |
| **Priority** | P2                  |

**Кроки:**

1. Увійти в систему та відкрити базу даних Trading Journal.
2. Створити нову BUTTON-властивість із назвою `"Закрити угоду"` та налаштувати дію `"Встановити Status = Closed"`.
3. Увімкнути прапорець `"Вимагати підтвердження"` для цієї кнопки.
4. Знайти в таблиці існуючу угоду зі статусом `"Open"` та натиснути кнопку `"Закрити угоду"` в її рядку.
5. Переконатися, що на екрані з'явилося діалогове вікно підтвердження дії.
6. Натиснути кнопку `"Підтвердити"`.
7. Спостерігати за значенням поля `Status` для обраного запису.
8. Перевірити, що статус успішно змінився на `"Closed"` в реальному часі.

**Очікуваний результат:**

- Діалогове вікно з'являється після натискання на кнопку.
- Після підтвердження статус запису оновлюється без перезавантаження сторінки.
