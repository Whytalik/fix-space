# Playwright UI E2E Tests: Templates

### [x] TC-TMPL-016: Створення та застосування кастомного шаблону запису (Playwright UI E2E)

| Поле         | Значення            |
| ------------ | ------------------- |
| **US**       | US-005              |
| **Issue**    | #87                 |
| **TS**       | TS-TMPL-01          |
| **Метод**    | E2E UI (Playwright) |
| **Техніка**  | Exploratory Testing |
| **Priority** | P2                  |

**Кроки:**

1. Увійти в систему та відкрити існуючу базу даних.
2. Відкрити сторінку Edit бази даних (`/database/:id/edit`)
3. У секції Templates натиснути "Add template"
4. Ввести назву нового шаблону `"Мій кастомний шаблон"` та зберегти
5. Перевірити, що шаблон з'явився у списку шаблонів

**Очікуваний результат:**

- Новий кастомний шаблон успішно створюється та з'являється у списку шаблонів.

### [x] TC-TMPL-017: Видалення шаблону через UI (Playwright)

| Поле         | Значення            |
| ------------ | ------------------- |
| **US**       | US-005              |
| **Issue**    | —                   |
| **TS**       | TS-TMPL-01          |
| **Метод**    | E2E UI (Playwright) |
| **Техніка**  | State Transition    |
| **Priority** | P2                  |

**Кроки:**

1. Відкрити сторінку Edit бази даних (`/database/:id/edit`)
2. Перейти до секції Templates та знайти наявний кастомний шаблон
3. Навести курсор на шаблон та клікнути іконку видалення або вибрати "Delete"
4. Підтвердити видалення у діалоговому вікні
5. Перевірити що шаблон зник зі списку та з template picker

**Очікуваний результат:**

- Шаблон видаляється зі списку в редакторі та з меню вибору шаблону при створенні запису
- Після оновлення сторінки шаблон відсутній
