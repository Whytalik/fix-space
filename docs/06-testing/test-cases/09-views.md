### [ ] TC-VIEW-029: Перемикання та відображення різних видів бази даних (Table, Board, Gallery) (Playwright UI E2E)

| Поле         | Значення            |
| ------------ | ------------------- |
| **US**       | US-072, US-082      |
| **Issue**    | #14, #67            |
| **TS**       | TS-REC-01           |
| **Метод**    | E2E UI (Playwright) |
| **Техніка**  | Exploratory Testing |
| **Priority** | P1                  |

**Кроки:**

1. Увійти в систему та відкрити базу даних з декількома записами.
2. Знайти панель вкладок видів (Views) зверху бази даних.
3. Натиснути кнопку "+ Додати вид" та обрати тип "Board" (Канбан-дошка) з групуванням по статусу, назвати `"Канбан"`.
4. Переконатися, що записи відображаються у вигляді карток у колонках відповідних статусів.
5. Натиснути кнопку "+ Додати вид" знову, обрати тип "Gallery" (Галерея), назвати `"Галерея"`.
6. Переконатися, що записи рендеряться у вигляді сітки карток з прев'ю.
7. Перемкнутися назад на дефолтну вкладку "Table" (Таблиця) та переконатися, що записи знову відображаються у вигляді рядків таблиці.

**Очікуваний результат:**

- Всі три види бази даних (Table, Board, Gallery) створюються та перемикаються без помилок в UI.
