### TC-IMP-011: Експорт бази даних у форматі CSV через меню опцій (Playwright UI E2E)

| Поле         | Значення            |
| ------------ | ------------------- |
| **US**       | US-089              |
| **Issue**    | #50                 |
| **TS**       | —                   |
| **Метод**    | E2E UI (Playwright) |
| **Техніка**  | Exploratory Testing |
| **Priority** | P2                  |

**Кроки:**

1. Увійти в систему та відкрити існуючу базу даних (наприклад, Trading Journal).
2. Натиснути на кнопку меню опцій бази даних (іконка з трьома крапками або шестірня у правому верхньому кутку).
3. Обрати пункт меню "Експортувати в CSV" (Export to CSV).
4. У вікні налаштування експорту вибрати всі колонки та підтвердити дію.
5. Зачекати завершення генерації та переконатися, що браузер розпочав і успішно завершив завантаження файлу з розширенням `.csv`.

**Очікуваний результат:**

- Браузер успішно завантажує файл CSV, що містить валідний звіт по записах бази даних.
