### TC-IMP-010: Звіт валідації — скасування або підтвердження

| Поле         | Значення         |
| ------------ | ---------------- |
| **US**       | US-090           |
| **Issue**    | #104             |
| **TS**       | —                |
| **Метод**    | Manual (Browser) |
| **Техніка**  | State Transition |
| **Priority** | P3               |

**Кроки:**

1. Отримати звіт з помилками
2. Натиснути "Cancel" — перевірити що імпорт скасовано
3. Повторити, натиснути "Import anyway" — перевірити що імпорт виконано

**Очікуваний результат:**

- Cancel: імпорт скасовано
- Import anyway: дані імпортовано (помилкові рядки пропущено)
