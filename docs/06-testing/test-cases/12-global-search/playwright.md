### [x] TC-SEARCH-006: Швидкий глобальний пошук за назвою запису через діалогове вікно (Playwright UI E2E)

| Поле         | Значення            |
| ------------ | ------------------- |
| **US**       | US-027, US-086      |
| **Issue**    | #47                 |
| **TS**       | —                   |
| **Метод**    | E2E UI (Playwright) |
| **Техніка**  | Exploratory Testing |
| **Priority** | P2                  |

**Кроки:**

1. Увійти в систему та переконатися, що у робочому просторі є запис із назвою `"Угода на золото XAUUSD"`.
2. Натиснути комбінацію клавіш `Ctrl+K` або натиснути кнопку пошуку на верхній панелі інтерфейсу.
3. Переконатися, що на екрані з'явилося модальне вікно глобального пошуку.
4. Ввести у текстове поле пошуку фразу `"золото"`.
5. Перевірити, що в списку результатів пошуку з'явився запис `"Угода на золото XAUUSD"` із підсвіченим збігом `"золото"`.
6. Натиснути клавішу `Enter` або клікнути на знайдений запис у списку.
7. Перевірити, що модальне вікно закрилося, а інтерфейс автоматично перейшов до перегляду/редагування обраного запису.

**Очікуваний результат:**

- Модальне вікно пошуку відкривається, шукає запис у реальному часі та перенаправляє користувача до знайденого об'єкта при виборі.

### [x] TC-SEARCH-007: Пошук за частиною назви запису через UI (Playwright)

| Поле         | Значення                              |
| ------------ | ------------------------------------- |
| **US**       | US-027                                |
| **Issue**    | —                                     |
| **TS**       | —                                     |
| **Метод**    | E2E UI (Playwright)                   |
| **Техніка**  | Equivalence Partitioning (valid path) |
| **Priority** | P2                                    |

**Кроки:**

1. Увійти в систему з наявними записами в базі даних
2. Відкрити глобальний пошук (іконка пошуку або Ctrl+K)

- Результати пошуку з'являються в реальному часі вже при введенні перших символів
- Клік на результат закриває пошук та переходить до відповідного запису
