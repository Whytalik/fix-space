## 3.20 TEXT

**Текстове поле** — зберігає рядок або форматований текст. Підтримує два режими: простий однорядковий рядок та rich text із розміткою. Налаштування `urlHandling` визначає поведінку URL у відображенні.

**Функціональні вимоги:**

- 📝 Додано до Issue (#60) Редагування в режимі inline — безпосередньо в комірці таблиці або полі картки запису
- 📝 Додано до Issue (#60) У режимі `isRichText: false` — однорядкове текстове поле
- 📝 Додано до Issue (#60) У режимі `isRichText: true` — відкривається rich text редактор: жирний, курсив, підкреслення, закреслення, код, посилання, виділення кольором
- 📝 Додано до Issue (#60) `urlHandling: detect` — URL автоматично перетворюються на клікабельні посилання при відображенні
- 📝 Додано до Issue (#60) `urlHandling: preview` — для підтримуваних URL (зображення, YouTube, інші сервіси) відображається вбудований попередній перегляд
- 📝 Додано до Issue (#60) Вставка через Ctrl+V зберігає форматування в режимі rich text; Ctrl+Shift+V — вставка без форматування
- 📝 Додано до Issue (#60) Порожнє значення зберігається як порожній рядок; null форматується як `""` при відображенні
- 📝 Додано до Issue (#60) Підтримує масове редагування (перезаписати значення для всіх вибраних записів)
