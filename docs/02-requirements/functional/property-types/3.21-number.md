## 3.21 NUMBER

**Числове поле** — зберігає числове значення з гнучким форматуванням відображення. `prefix`, `suffix` та `currencySymbol` впливають лише на те, як число виглядає — збережене значення завжди є чистим числом.

**Функціональні вимоги:**

- Редагування в режимі inline; відхиляє нечислові символи
- Формат `integer`: десяткова крапка заблокована; при введенні нецілого значення округлюється автоматично
- Формат `currency`: `currencySymbol` відображається перед числом (наприклад, `$150.00`)
- Формат `percentage`: значення відображається з суфіксом `%`
- `prefix` і `suffix` відображаються навколо числа в таблиці та картці (наприклад, `2.5 R`, `0.1 lots`, `×5`)
- `decimalPlaces` визначає кількість знаків після коми при відображенні; збережене значення зберігається з повною точністю
- Порожнє значення відображається як порожня комірка; null форматується як `0` при обчисленнях у формулах та підсумках
- Підтримує масове редагування
