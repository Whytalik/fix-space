## 3.29 PROGRESS

**Поле прогресу** — числове значення у заданому діапазоні `[min, max]`. Відображається як горизонтальний прогрес-бар із опційним числовим значенням поруч.

**Функціональні вимоги:**

- 📝 Додано до Issue (#69) Відображається як горизонтальний бар заповнення; відсоток заповнення = `(value - min) / (max - min) × 100%`
- 📝 Додано до Issue (#69) Якщо `showLabel: true` — числове значення відображається поруч із баром (праворуч або під баром залежно від ширини комірки)
- 📝 Додано до Issue (#69) Редагування: клік на комірку відкриває slider та числове поле введення
- 📝 Додано до Issue (#69) Drag по slider або введення числа вручну; значення обрізається до `[min, max]`
- 📝 Додано до Issue (#69) Крок зміни slider відповідає `step`; клавіші ←/→ змінюють значення на один `step`
- 📝 Додано до Issue (#69) Колір бару визначається масивом `thresholds`: перевіряються зліва направо, застосовується перший поріг, для якого `value ≤ upTo`; якщо жоден не підходить — використовується останній поріг; при порожньому `thresholds` — акцентний колір поточної теми
- 📝 Додано до Issue (#69) null відображається як порожній бар (0% заповнення)
- 📝 Додано до Issue (#69) Підтримує масове редагування
