## 3.28 RATING

**Поле рейтингу** — числова оцінка у форматі зірок. Значення від 0 до `maxStars`. Використовується для суб'єктивної оцінки якості угоди, дня, рішення тощо.

**Функціональні вимоги:**

- Відображається як горизонтальний ряд зірок (`★` заповнені / `☆` порожні); загальна кількість = `maxStars`
- Клік на зірку встановлює рейтинг, рівний позиції зірки (1–`maxStars`)
- Повторний клік на ту саму зірку скидає рейтинг до 0 (очищення)
- Hover-стан: підсвічуються всі зірки від першої до наведеної включно
- Якщо `allowHalf: true` — hover враховує половину зірки: ліва половина зірки = `N - 0.5`, права = `N`; відображається `★½` для дробових значень
- Значення обрізається до діапазону `[0, maxStars]` при збереженні
- null відображається як порожній ряд (0 заповнених зірок)
- Підтримує масове редагування
