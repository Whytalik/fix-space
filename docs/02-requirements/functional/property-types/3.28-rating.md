## 3.28 RATING

**Поле рейтингу** — числова оцінка у форматі зірок. Значення від 0 до `maxStars`. Використовується для суб'єктивної оцінки якості угоди, дня, рішення тощо.

**Функціональні вимоги:**

- 📝 Додано до Issue (#68) Відображається як горизонтальний ряд зірок (`★` заповнені / `☆` порожні); загальна кількість = `maxStars`
- 📝 Додано до Issue (#68) Клік на зірку встановлює рейтинг, рівний позиції зірки (1–`maxStars`)
- 📝 Додано до Issue (#68) Повторний клік на ту саму зірку скидає рейтинг до 0 (очищення)
- 📝 Додано до Issue (#68) Hover-стан: підсвічуються всі зірки від першої до наведеної включно
- 📝 Додано до Issue (#68) Якщо `allowHalf: true` — hover враховує половину зірки: ліва половина зірки = `N - 0.5`, права = `N`; відображається `★½` для дробових значень
- 📝 Додано до Issue (#68) Значення обрізається до діапазону `[0, maxStars]` при збереженні
- 📝 Додано до Issue (#68) null відображається як порожній ряд (0 заповнених зірок)
- 📝 Додано до Issue (#68) Підтримує масове редагування
