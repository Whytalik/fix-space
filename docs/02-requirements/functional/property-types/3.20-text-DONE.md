## 3.20 TEXT

**Текстове поле** — зберігає рядок або форматований текст. Завжди працює в режимі Rich Text з підтримкою базового форматування та автоматичним розпізнаванням посилань.

**Функціональні вимоги:**

- [x] Відкривається rich text редактор: жирний, курсив, підкреслення (через bubble menu при виділенні тексту)
- [x] URL автоматично перетворюються на клікабельні посилання при відображенні
- [x] Посилання завжди підкреслені, мають колір тексту (не сині), при відвідуванні колір стає тьмянішим
- [x] Вставка через Ctrl+V зберігає форматування; Ctrl+Shift+V — вставка без форматування
- [x] Підтримка гарячих клавіш: Ctrl+B (жирний), Ctrl+I (курсив), Ctrl+U (підкреслення)
- [x] Порожнє значення зберігається як порожній рядок; null форматується як `""` при відображенні

**Параметри конфігурації:**

| Параметр       | Тип    | Допустимі значення | За замовчуванням |
| -------------- | ------ | ------------------ | ---------------- |
| `defaultValue` | string | Будь-який рядок    | `""`             |

**Значення:** рядок (HTML) або null (null → порожній рядок).

**Оператори фільтрації:** `equals`, `not_equals`, `contains`, `not_contains`, `starts_with`, `ends_with`, `is_empty`, `is_not_empty`
