## 3.24 DURATION

**Поле тривалості** — зберігає ціле невід'ємне число секунд. Відображається у форматі часу, обраному в конфізі. Використовується для часу утримання позиції, тривалості сесії тощо.

**Функціональні вимоги:**

- 📝 Додано до Issue (#64) Редагування через inline-поле з розпізнаванням введеного формату:
  - `1:30` → 5 400 с (інтерпретується як `HH:MM`)
  - `1:30:00` → 5 400 с (`HH:MM:SS`)
  - `90` → 90 с (якщо активний формат `minutes` або `seconds`)
- 📝 Додано до Issue (#64) Від'ємні значення та нечислові символи відхиляються; порожній ввід зберігається як null
- 📝 Додано до Issue (#64) Відображення у таблиці та картці згідно з `format` властивості:
  - `HH:mm` → `01:30`
  - `HH:mm:ss` → `01:30:00`
  - `Xh Ym` → `1h 30m`
  - `minutes` → `90`
  - `seconds` → `5400`
- 📝 Додано до Issue (#64) null відображається як `0` / `00:00` залежно від формату
- 📝 Додано до Issue (#64) Підтримує масове редагування
