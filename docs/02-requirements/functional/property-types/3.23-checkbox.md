## 3.23 CHECKBOX

**Поле-прапорець** — зберігає булеве значення: відмічено або не відмічено. Найпростіший тип введення — один клік без відкриття редактора.

**Функціональні вимоги:**

- 📝 Додано до Issue (#63) Перемикач кліком безпосередньо в комірці таблиці або картці запису — редактор не відкривається
- 📝 Додано до Issue (#63) Відображається як квадратний чекбокс; `true` — заповнений з галочкою, `false` / null — порожній
- 📝 Додано до Issue (#63) null відображається як `false`; збережене значення залишається null до першого явного встановлення
- 📝 Додано до Issue (#63) Якщо властивість позначена як `isRequired: true` — запис не можна зберегти, поки значення явно не встановлено в `true`
- 📝 Додано до Issue (#63) Підтримує масове редагування: встановити всі вибрані в `true` або `false`
