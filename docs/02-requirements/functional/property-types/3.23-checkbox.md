## 3.23 CHECKBOX

**Поле-прапорець** — зберігає булеве значення: відмічено або не відмічено.

**Функціональні вимоги:**

- [x] Відображається як квадратний чекбокс; `true` — заповнений з галочкою, `false` — порожній
- [x] Значення є строго логічним (`true` або `false`). Порожніх (`null`) значень для чекбокса не існує на рівні відображення та збереження — відсутні дані трактуються як `false`

**Параметри конфігурації:**

| Параметр       | Тип     | Допустимі значення | За замовчуванням |
| -------------- | ------- | ------------------ | ---------------- |
| `defaultValue` | boolean | true / false       | `false`          |

**Значення:** `true` або `false`.

**Оператори фільтрації:** `is_checked` (true), `is_unchecked` (false)
