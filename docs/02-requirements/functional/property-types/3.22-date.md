## 3.22 DATE

**Поле дати** — зберігає дату або дату з часом у форматі ISO 8601 (UTC). Відображає у форматі та часовому поясі з налаштувань користувача.

**Функціональні вимоги:**

- [ ] Редагування через календарний попап зі стрілками навігації по місяцях
- [ ] Якщо `includeTime: true` — поруч із календарем відображається time picker у форматі `HH:mm` або `hh:mm A` (відповідно до `timeFormat`)
- [ ] Кнопка «Сьогодні» в попапі встановлює поточну дату (враховує часовий пояс із налаштувань користувача)
- [ ] `defaultValue: "today"` замінюється поточною датою в момент **створення запису**, а не при відкритті форми
- [ ] Очищення значення — кнопка «Очистити» в попапі або клавіша Delete у виділеній комірці
- [ ] Відображення у таблиці: формат із конфіга властивості; якщо `includeTime: false` — час не відображається навіть якщо збережений у значенні
- [ ] Відносне відображення («Сьогодні», «Вчора», «2 дні тому») — опційне налаштування подання
- [ ] Підтримує масове редагування

**Параметри конфігурації:**

| Параметр       | Тип            | Допустимі значення                         | За замовчуванням |
| -------------- | -------------- | ------------------------------------------ | ---------------- |
| `defaultValue` | string \| null | ISO-рядок · `"today"` · null               | `null`           |
| `format`       | enum           | `DD.MM.YYYY` · `MM/DD/YYYY` · `YYYY-MM-DD` | `"DD.MM.YYYY"`   |
| `includeTime`  | boolean        | true / false                               | `true`           |
| `timeFormat`   | enum           | `HH:mm` · `hh:mm A`                        | `"HH:mm"`        |

Спеціальне значення `"today"` замінюється поточною датою на момент створення запису.

**Значення:** рядок ISO 8601 або null.

**Оператори фільтрації:** `equals`, `not_equals`, `before`, `after`, `on_or_before`, `on_or_after`, `is_empty`, `is_not_empty`
