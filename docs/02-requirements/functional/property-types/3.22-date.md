## 3.22 DATE

**Поле дати** — зберігає дату або дату з часом у форматі ISO 8601 (UTC). Відображає у форматі та часовому поясі з налаштувань користувача.

**Функціональні вимоги:**

- 📝 Додано до Issue (#62) Редагування через календарний попап зі стрілками навігації по місяцях
- 📝 Додано до Issue (#62) Якщо `includeTime: true` — поруч із календарем відображається time picker у форматі `HH:mm` або `hh:mm A` (відповідно до `timeFormat`)
- 📝 Додано до Issue (#62) Кнопка «Сьогодні» в попапі встановлює поточну дату (враховує часовий пояс із налаштувань користувача)
- 📝 Додано до Issue (#62) `defaultValue: "today"` замінюється поточною датою в момент **створення запису**, а не при відкритті форми
- 📝 Додано до Issue (#62) Очищення значення — кнопка «Очистити» в попапі або клавіша Delete у виділеній комірці
- 📝 Додано до Issue (#62) Відображення у таблиці: формат із конфіга властивості; якщо `includeTime: false` — час не відображається навіть якщо збережений у значенні
- 📝 Додано до Issue (#62) Відносне відображення («Сьогодні», «Вчора», «2 дні тому») — опційне налаштування подання
- 📝 Додано до Issue (#62) Підтримує масове редагування
