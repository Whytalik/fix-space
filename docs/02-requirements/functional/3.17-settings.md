## 3.17 Налаштування <!-- US-120, US-121, US-122, US-123, US-124, US-009, US-004, US-007, US-008, US-012, US-013, US-014, US-052 -->

**Налаштування** — це централізована сторінка управління параметрами облікового запису, інтерфейсу та значень за замовчуванням для об'єктів системи. Сторінка організована у вісім незалежних груп, розподілених за логічним рівнем: особистий акаунт, відображення інтерфейсу, об'єкти системи та зовнішні підключення. <!-- US-120, US-121, US-122, US-123, US-124, US-009 -->

**Профіль** — персональні дані облікового запису та управління його існуванням.

**Безпека** — управління засобами автентифікації та активними сесіями. <!-- US-004, US-007, US-008, US-124 -->

**Зовнішній вигляд** — глобальні параметри відображення інтерфейсу, незалежні від конкретного простору. <!-- US-120, US-121 -->

**Робочий простір** — параметри відображення даних, що застосовуються в межах усіх просторів: формат дат і часу, початок тижня. Ці налаштування впливають на те, як відображаються значення в базах даних, а не на візуальний стиль інтерфейсу. <!-- US-122 -->

**Секції** — значення за замовчуванням, що застосовуються при створенні нової секції. <!-- US-123 -->

**Бази даних** — значення за замовчуванням, що застосовуються при створенні нової бази даних. <!-- US-123 -->

**Записи** — значення за замовчуванням, що застосовуються при створенні нового запису. <!-- US-123, US-052 -->

**Інтеграції** — управління підключеннями до зовнішніх брокерських сервісів (детальна специфікація — розділ 3.12). <!-- US-091, US-092, US-093 -->

**Функціональні вимоги:**

Профіль:

- [ ] Зміна аватару (завантаження нового зображення, видалення поточного) <!-- US-120 -->
- [ ] Зміна нікнейму <!-- US-120 -->
- [ ] Зміна електронної пошти облікового запису (потрібне підтвердження через лист на нову адресу) <!-- US-120 -->
- [ ] Видалення облікового запису (потрібне підтвердження паролем; всі дані видаляються безповоротно; підтвердження надсилається на email) <!-- US-009 -->

Безпека:

- [ ] Зміна пароля (потрібен поточний пароль; після зміни всі активні сесії анулюються) <!-- US-004 -->
- [ ] Перегляд списку активних сесій (IP-адреса, пристрій, браузер, приблизне місцезнаходження, дата входу) <!-- US-007 -->
- [ ] Можливість завершити конкретну сесію (дистанційний вихід) <!-- US-124 -->
- [ ] Кнопка «Вийти з усіх пристроїв» (анулює всі сесії, крім поточної) <!-- US-008 -->

Зовнішній вигляд:

- [ ] Вибір теми оформлення: Світла / Темна / Як у системі <!-- US-120 -->
- [ ] Вибір мови інтерфейсу: uk / en <!-- US-121 -->

Робочий простір:

- [ ] Вибір формату відображення дат
- [ ] Вибір формату відображення часу: 12-годинний / 24-годинний
- [ ] Вибір першого дня тижня: понеділок / неділя
- [ ] Вибір часового поясу (IANA-формат, наприклад Europe/Kyiv, America/New_York); за замовчуванням — автовизначення з браузера при реєстрації; усі дати та часові мітки відображаються у вибраному поясі, зберігаються в UTC
- [ ] Перейменування та видалення наявних робочих просторів (видалення основного простору заборонено)
- [ ] Зміна основного робочого простору (встановити інший простір як основний)

Секції:

- [ ] Вибір іконки та кольору за замовчуванням для нових секцій
- [ ] Увімкнення або вимкнення діалогу підтвердження перед видаленням секції

Бази даних:

- [ ] Вибір іконки за замовчуванням для нових баз даних

Записи:

- [ ] Вибір іконки за замовчуванням для нових записів
- [ ] Автоматичне відкриття нового запису для редагування після його створення

Інтеграції:

- [ ] Перелік усіх налаштованих підключень до зовнішніх сервісів із відображенням статусу кожного
- [ ] Додавання нового підключення
- [ ] Перепідключення (оновлення credentials) наявного підключення
- [ ] Відключення наявного підключення (з підтвердженням)
