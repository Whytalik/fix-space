## 3.18 Домашня сторінка

- [x] Кожен простір має **домашню сторінку** — окрему сторінку-дашборд, доступну через кнопку у сайдбарі. Вона є точкою входу до простору та відкривається автоматично при перемиканні між просторами.
- [x] Домашня сторінка складається з чотирьох функціональних блоків:
- [x] 1. **Індикатор ринкових сесій** — відображає поточний час, день тижня та активні торгові сесії (Токіо, Франкфурт, Лондон, Нью-Йорк) із кольоровими індикаторами, а також час відкриття наступної сесії.
- [x] 2. **Щоденний воркфлоу** — горизонтальна панель із 4 кроками послідовного процесу трейдера: «Plan Routine -> Execute Trade -> Reflect Notes -> Log Mistakes». Кожен крок відображає статус виконання потового дня (виконано / не виконано). Натискання на крок відкриває нову сторінку запису у відповідній базі даних.
- [x] 3. **Сьогодні** — 4 картки (Trading Journal, Daily Routine, Notes, Mistakes) із переліком записів, створених за поточий день, та кнопкою швидкого додавання.
- [x] 4. **Огляд** — 3 mini-графіки: P&L Curve (динаміка прибутку), Win Rate Dynamics (динаміка відсотку прибуткових угод), RR Deviation (відхилення фактичного RR від запланованого). (Реалізовано структуру API та компонентів, візуалізація потребує інтеграції бібліотеки графіків)

**Функціональні вимоги:**

- [x] Перегляд домашньої сторінки (дашборду) поточного простору
- [x] Перехід до домашньої сторінки через кнопку у сайдбарі
- [x] Відображення поточного часу та активних торгових сесій у режимі реального часу
- [x] Відображення статусу виконання щоденного воркфлоу (чи був створений хоча б один запис сьогодні в кожній із 4 баз)
- [x] Швидке створення запису з домашньої сторінки (перехід до бази та відкриття нового запису в режимі редагування)
- [x] Відображення записів, створених за поточий день, у секції «Сьогодні»
- [x] Відображення міні-графіків поточної торгової статистики (API підготовлено, UI інтегрує заглушку)
