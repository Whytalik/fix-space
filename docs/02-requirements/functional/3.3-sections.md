## 3.3 Секції

- [ ] **Секція** — це іменована група баз даних усередині робочого простору. Вона не зберігає записи і не має власних налаштувань даних — її єдина функція полягає в організації навігаційної структури сайдбару.
- [ ] Секції залишаються необхідним рівнем організації незалежно від кількості баз даних: зі зростанням їх числа плоский список стає незручним для навігації, тоді як секції дозволяють зберігати чітку структуру. Можливість згортати секції в сайдбарі зменшує когнітивне навантаження під час активної торгової сесії — видно лише те, що потрібно зараз.

**Функціональні вимоги:**

- [ ] Автоматичне створення стартових секцій при ініціалізації робочого простору
- [ ] Створення нової секції
- [ ] Редагування назви, іконки та кольору секції
- [ ] Зміна порядку секцій у сайдбарі
- [ ] Переміщення бази даних між секціями
- [ ] Зміна порядку баз даних усередині секції
- [ ] Видалення секції (бази даних із видаленої секції залишаються в просторі без прив'язки до секції та відображаються в сайдбарі нижче всіх секцій)
- [ ] Згортання та розгортання секції в сайдбарі
- [ ] Збереження стану згортання кожної секції між сесіями
