## 3.3 Секції

- [x] **Секція** — це іменована група баз даних усередині робочого простору. Вона не зберігає записи і не має власних налаштувань даних — її єдина функція полягає в організації навігаційної структури сайдбару.
- [x] Секції залишаються необхідним рівнем організації незалежно від кількості баз даних: зі зростанням їх числа плоский список стає незручним для навігації, тоді як секції дозволяють зберігати чітку структуру. Можливість згортати секції в сайдбарі зменшує когнітивне навантаження під час активної торгової сесії — видно лише те, що потрібно зараз.

**Функціональні вимоги:**

- [x] Автоматичне створення стартових секцій при ініціалізації робочого простору
- [x] Створення нової секції
- [x] Редагування назви, іконки та кольору секції
- [x] Зміна порядку секцій у сайдбарі
- [x] Переміщення бази даних між секціями
- [x] Зміна порядку баз даних усередині секції
- [x] Видалення секції (бази даних із видаленої секції залишаються в просторі без прив'язки до секції та відображаються в сайдбарі нижче всіх секцій)
- [x] Згортання та розгортання секції в сайдбарі
- [x] Збереження стану згортання кожної секції між сесіями
